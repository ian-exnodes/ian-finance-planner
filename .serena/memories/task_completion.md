# Task Completion Memory

A task is complete only when:

- Requested behavior is implemented.
- Visible UI copy is Vietnamese.
- Data access respects Supabase auth and RLS.
- Relevant calculations handle edge cases.
- TypeScript/build checks pass, or failures are clearly documented.
- `docs/05-TASKLIST.md` is updated.
- Durable knowledge is saved to Serena memory if changed.
- A checkpoint is written for significant sessions.
