# Core Memory

- Product: personal finance debt planner.
- User: one private individual.
- UI language: Vietnamese only.
- Developer-facing docs/code may use English.
- Backend: Supabase.
- Deployment: Vercel.
- Main goal: show how much money must be reserved each month for repayments and whether planned purchases are financially safe.
- v1 is not a team app, bank-sync app, accounting app, or public SaaS.
