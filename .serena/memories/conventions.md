# Conventions Memory

- Visible UI must be Vietnamese.
- Developer-facing code names may be English.
- Currency format: Vietnamese VND, no decimals.
- Month display: `Tháng MM/YYYY`.
- Date display: `DD/MM/YYYY`.
- Use React Hook Form + Zod for forms.
- Use shadcn/ui before creating custom primitives.
- Confirmation dialog required before destructive deletes.
- Keep UX calm and non-judgmental.
